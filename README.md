# the-aios.org

The **Guardian / commons** surface for [The AIOS](https://the-aios.com) — plus the plugin & services **marketplace**.

- `/` → **the-aios.org** — the commons: the open framework, its license, and its name, kept free and held in trust.
- `/plugins` → **the marketplace** — plugins, credentials, and extensions from the ecosystem; where vendors and coaches set up shop.

Both are branded placeholder heroes for now (The-AIOS product brand: coral `#ff5d4d` · Inter · deep black).

## Deploy

Static site — no build step. Connect to Vercel (framework preset: **Other**), point the apex `the-aios.org` (+ `www`) at it. `git push origin main` → auto-deploy.
